Application de Vote Électronique

Contexte

Développement d'une application client-serveur sécurisée pour le vote électronique, utilisant le chiffrement ElGamal et une interface en JavaFX. Ce projet met l'accent sur la sécurité cryptographique et l'intégrité des données de vote.

Technologies utilisées

  • Java (Backend & Frontend)
  • JavaFX (Interface graphique)
  • Sockets Sécurisées (SSL/TLS)
  • Chiffrement ElGamal
  • Cryptographie asymétrique

Sécurité implémentée

  • Chiffrement homomorphe ElGamal
  • Génération de clés cryptographiques
  • Authentification des votants
  • Protection contre la double-vote
  • Anonymat des bulletins

Architecture du système

Composants principaux

  • Serveur de vote centralisé
  • Client de vote JavaFX
  • Module de chiffrement/déchiffrement
  • Base de données sécurisée
  • Interface d'administration

Apprentissages

  • Implémentation du chiffrement ElGamal
  • Développement d'interfaces graphiques avec JavaFX
  • Architecture client-serveur robuste
  • Sécurisation des données sensibles
  • Gestion des grands nombres en cryptographie

Démonstration du système de vote

Cette application de vote électronique utilise le chiffrement ElGamal pour garantir :

  • Confidentialité : Les votes sont chiffrés individuellement
  • Intégrité : Impossible de modifier les votes sans détection
  • Anonymat : Aucun lien entre l'identité du votant et son choix
  • Vérifiabilité : Possibilité de vérifier le bon décompte
