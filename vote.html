Application de Vote Électronique

Contexte

Développement d'une application client-serveur sécurisée pour le vote électronique, utilisant le chiffrement ElGamal et une interface en JavaFX.

Technologies utilisées

  • Java
  • JavaFX
  • Sockets Sécurisées
  • Chiffrement ElGamal

Apprentissages

  • Implémentation du chiffrement ElGamal
  • Développement d'interfaces graphiques avec JavaFX
  • Architecture client-serveur
  • Sécurisation des données sensibles
